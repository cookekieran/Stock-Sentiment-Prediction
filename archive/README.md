# Archive

This directory preserves superseded experiments and project-history notes.

- `legacy_experiments`: early FinBERT, FinGPT, DeepSeek, notebook, and saved
  single-run model artifacts, including the superseded GRU pipelines and their
  generated model outputs.
- `project_history`: chronological decision logs and conversation summaries.

Archived work is retained for dissertation traceability but is not part of the
canonical final-study workflow described in the root README.
